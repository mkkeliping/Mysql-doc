# Vim文本编辑器
## Vim介绍
*Vim是一个类似于Vi的著名的功能强大、高度可定制的文本编辑器，在Vi的基础上改进和增加了很多特性。Vim是纯粹的自由软件在代码补全、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用，和Emacs并列成为类Unix系统用户最喜欢的文本编辑器*
## Vim安装
*执行语句sudo apt-get install vim*
*接着会出现如图界面，输入您的密码，等待安装，安装过程中会出现如下界面，直接按回车即可*
## Vim基本操作
1. 操作玩进入编辑状态
    i：在当前字符左侧插入
    I：当前行首插入
    a: 当前字符右侧插入      
    A：当前行尾插入
    o: 当前行下一行插入
    O：当前行上一行插入
