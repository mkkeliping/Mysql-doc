## 乱码
乱码有两种，写入数据库时乱码，从数据库读出乱码
>乱码
>>解决方法                                         
方法一：创建数据库时和基本表时设置character set = utf8;                         
方法二：在.c中设置mysql_options(db,MYSQL_SET_CHARSET_NAME,"utf8");                         
方法三：mysql> show variables like 'character%'
